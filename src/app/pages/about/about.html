About Shobaby

Shobaby is a modern e-commerce platform built with Angular and Tailwind CSS. Our mission is to provide a seamless shopping experience with a clean, intuitive interface.

Our Features

  • Browse a wide range of products from various categories
  • Detailed product information with ratings and reviews
  • User-friendly shopping cart experience
  • Secure user authentication
  • Responsive design that works on all devices
  • Dark mode support for comfortable browsing

Technology Stack

  • Angular 20 with standalone components
  • Tailwind CSS for styling
  • RxJS for reactive programming
  • Fake Store API for product data
